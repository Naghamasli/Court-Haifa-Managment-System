A full-scale Java Object-Oriented application designed to manage court operations, including cases, judges, lawyers, hearings, and legal documents.
Built with a strong focus on clean architecture, scalability, and real-world system design.

🏗️ System Architecture

The project follows a clean MVC pattern:

Model (Entity) → Represents core data (Case, Judge, Lawyer…)
View (Boundary) → Java Swing UI screens
Controller (Logic/Control) → Handles business logic and operations

This ensures:

Maintainability
Readability
Easy future expansion
